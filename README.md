# Presales Q1 2026 Review

Interactive web presentation for the Elision Technologies Presales Q1 business review.

## View locally

Open `index.html` in a modern browser. An internet connection is required for Tailwind CSS, Chart.js, and Google Fonts.

## Navigation

- Use the expandable menu on the left to select a slide.
- Use the left/right arrow keys or Page Up/Page Down.
- Press Space to move to the next slide.

## Data source

Pipedrive.
